About

Detail-oriented by necessity, not by trend.

I'm Louie Cagas, a full-stack web developer based in Gingoog City, Misamis Oriental. My work centers on PHP, Laravel, CodeIgniter and WordPress — building web applications that real businesses depend on, not demos that only work in a pitch.

Most recently that's meant designing an all-in-one ERP system spanning inventory, sales, HR and resource management, and integrating third-party APIs so separate tools actually talk to each other. Before that, I kept a company's servers, point-of-sale software and internal tools running day to day — so I understand what "in production" really demands.

I care about the parts clients don't see until something breaks: server-side validation, clean deployments, and code that the next developer can actually read.
